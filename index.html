    Was ist SwordCraftOnline Überhaupt?

    Im Folgenden, findest du eine kurze Einleitung zu unserem Server :)

    Unser Spielmodus basiert auf dem Anime SwordArtOnline, wodurch unser Server auch diesen Namen erhielt. Da wir Sword Art Online nicht 1 - zu 1 Kopieren wollten, haben wir unser eigenes Spielprinzip ins leben gerufen welches dem aus SAO jedoch immernoch sehr stark ähnelt. Sollte man in SwordCraftOnline sterben, kann man für die nächten 2-5 Minuten nicht auf den Server. Zudem bieten wir viele eigens entwickelte Features ,wie Schwerter die einem z.b Sprungkraft geben, dazu verfügen wir über ein Skill System, wo man die unterschiedlichsten Skills erlernen und nutzen kann... Von Geschwindigkeit bis zum Inventar Platz - Alles ist möglich und ein Langer Spielspaß ist garantiert ! Wir bieten unseren Usern ganze 20 Ebenen (welche pro Ebene eine Zeit von cirka. 5 Stunden in Anspruch nehmen).

    Viel Spaß auf unserem SwordCraftOnline.de Server wünscht dir dein SCO Team! (Sollten weitere Fragen auftauchen wende dich bitte an das Server Team)
